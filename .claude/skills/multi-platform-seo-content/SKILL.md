# Multi-Platform SEO Content Generator

Generate fully SEO-optimized posting content for Pinterest, Instagram/Facebook, TikTok, and YouTube.

## Input Requirements

- **Target Keyword** (required): The main keyword to optimize for
- **Product / Topic** (required): The product or topic to create content about
- **Target Country** (optional): Default is USA
- **Tone** (optional): Default is professional, friendly

## Workflow

1. **Interpret search intent** (buyer vs informational) from the keyword
2. **Adapt wording** to each platform's algorithm and user behavior
3. **Keep content natural**, avoid spam patterns
4. **Respect platform limits** as specified below

## Output Format

Return results in clearly separated sections per platform.

### PINTEREST

**Title:**
- 40-100 characters
- Start with the main keyword

**Description:**
- 300-500 characters
- Keyword in first 2 lines
- Include 2-3 secondary keywords naturally

**Hashtags:**
- 3-5 relevant hashtags

**Image Alt Text:**
- 100-200 characters
- Describe the image + include keyword naturally

### INSTAGRAM / FACEBOOK

**Caption:**
- 150-300 characters
- First line = hook with keyword
- Include a soft CTA at the end

**Hashtags:**
- 5-10 niche + buyer-intent hashtags

**Alt Text:**
- 100-150 characters
- Plain English image description

### TIKTOK

**Caption:**
- 80-150 characters
- Written like a search query
- Keyword in first sentence

**Hashtags:**
- 3-5 hashtags (1 broad, 2-3 niche)

**On-Screen Text:**
- 3-7 words
- Clear, readable, keyword-focused

**Alt Text:**
- 80-120 characters
- Describe the video scene and product

### YOUTUBE

**Title:**
- 60-90 characters
- Keyword near the beginning
- Benefit-driven

**Description:**
- 150-300 words
- First 2 lines are SEO-critical
- Include keyword variations naturally

**Tags:**
- 10-15 SEO tags
- Include exact keyword, variations, and product terms

**Hashtags:**
- 3 hashtags
- Exact-match keyword first

## Rules

- Do NOT reuse identical titles across platforms
- Avoid keyword stuffing
- Prioritize clarity, relevance, and click-through
- Output must be ready to post without editing
